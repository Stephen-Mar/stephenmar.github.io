# Stephen Mar — Personal Brand Website

A premium, minimalist personal brand site for Stephen Mar — entrepreneur and investor in crypto, DeFi, and Web3.

## Features
- Soft off-white + warm tan color palette (calm and luxurious, not overly bright)
- Perfectly fitted circular headshot in hero (full visibility, soft shadow, no cropping or edges)
- Entirely new professional layout: spacious hero, refined sections, masonry-style notes
- High-end typography, generous whitespace, and subtle transitions
- Fully responsive, Tailwind-powered, zero-build deployment

## GitHub Pages Deployment
1. Upload `index.html`, the `assets/images/` folder with `headshot.jpg`, and `README.md`.
2. Settings → Pages → Deploy from main branch → root folder.
3. Site live at `https://<username>.github.io/<repo-name>`.

## Customization
- Confirm exact path: `assets/images/headshot.jpg`
- Update notes, copy, or accents directly in the HTML.
