# Stephen Mar — Personal Brand Website

A clean, light-themed, minimalist single-page website for Stephen Mar, entrepreneur and investor focused on crypto, DeFi, and Web3.

## Features
- Bright white base with warm tan/gold accents for a premium, eye-pleasing aesthetic
- Single perfect circular headshot (`assets/images/headshot.jpg`) — fully visible, no cropping or white edges
- Spacious, modern layout with generous whitespace and elegant cards
- Real-time "Recent Notes" section
- Fully responsive with smooth navigation

## Deployment (GitHub Pages)
1. Create/upload files to your repository: `index.html`, `assets/images/headshot.jpg`, and `README.md`.
2. Go to **Settings → Pages** → Deploy from main branch → / (root).
3. Your site will be live shortly.

## Customization
- Ensure the image path is exactly `assets/images/headshot.jpg`.
- Edit notes, text, or accents directly in `index.html` for ongoing updates.
