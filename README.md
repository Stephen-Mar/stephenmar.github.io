# Stephen Mar — Personal Brand Website

A premium, calm, light-tan personal brand site for Stephen Mar — entrepreneur and investor in crypto, DeFi, and Web3.

## Features
- Soft off-white + warm tan palette (not bright)
- Entirely new professional layout and arrangement
- Dedicated `styles.css` for clean, maintainable premium styling
- Circular headshot (`assets/images/headshot.jpg`) that now fits perfectly with no cropping
- Responsive, smooth, and high-end feel

## Deployment (GitHub Pages)
1. Upload `index.html`, `styles.css`, the `assets/images/headshot.jpg` folder, and `README.md`.
2. Settings → Pages → Deploy from main branch → / (root).
3. Hard-refresh (Ctrl/Cmd + Shift + R) after deploy.
